PROJECT-1

TASK MANAGER APP

  • A simple to-do list app using JavaScript and local storage.
  • Features include add/delete tasks and save data on refresh.

USES

  • To-do lists help break down large goals into smaller, manageable tasks, making it easier to track progress and avoid feeling overwhelmed.
  • To-do lists help in estimating the time required for each task, allowing for better time allocation and planning.
